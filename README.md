# Minesweeper
[![Build Status](https://travis-ci.org/yuenm18/minesweeper.svg?branch=master)](https://travis-ci.org/yuenm18/minesweeper)

An online version of [minesweeper](https://yuenm18.github.io/minesweeper)

## Scripts

* Install - `npm install`
* Start - `npm start`
* Serve - `npm run serve`
* Lint - `npm run lint`
* Build - `npm run build`
