A [minesweeper](https://yuenm18.github.io/minesweeper) web application.

![screenshot](images/screenshot.png)

## Installation

1. Run `npm install`
2. Run `npm start`
3. Navigate to [http://localhost:8080](http://localhost:8080)

## Testing

1. Run `npm test`

## Linting

1. Run `npm run lint`
